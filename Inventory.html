Overview

System Goals

For this project, I wanted to learn more about RPG-style inventory systems in Unreal Engine and how to showcase my UI design skills during gameplay, and it ended up being a lot of fun to make. Here is a collection of the systems I have made so far:

  • Player item HUD: Displays player health, stamina, and items.
  • Interaction system for items, chests, and world objects.
  • Inventory menu with armor, consumables, and ammo equipment.
  • Item hover descriptions, sorting, dropping, and chest transfer.
  • Campfire rest system, potion healing, and bow + arrow combat support.
  • Icon design/implementation and locomotion-synced equipment.
