UI Test Project

Role: UI Designer, Programmer
Duration: 5 Hours (Iteration 1)
Engine: Unreal Engine 5 (CommonUI)

This is the first iteration of my UI Test Project, in which I will be learning more about CommonUI in Unreal Engine 5, creating a dynamic user interface for my game. Here is what I accomplished in my first 5 hours:

  • Live Main Menu Background
  • Pause Menu
  • Quit Game pop up with back input action functionality
  • Credits Menu
  • Dynamic context menu banner that switches button icons based on current input
  • KBM and Controller compatibility
  • Custom Mouse Cursor Design
  • Dynamic Menu transitions
  • Custom Button Designs with Animations
  • Appropriate UI SFX for selection

Next iteration will have a number of updates, including tab switching and a working settings menu. Stay tuned for more.
